# uv-activity-bzan545

Small practice repo for the BZAN 545 uv activity.

## Try it

```bash
uv sync
uv run python hello_orders.py
```

The `.venv/` folder is your local Python environment. It should appear on your
machine after uv sets up the project, but it should not be committed to Git.

During the activity, you will make a small edit to `hello_orders.py`, add a new
import, and use `uv add` to install the package needed by that import.
